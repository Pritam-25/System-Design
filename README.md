## 🚀 Interview Preparation Strategy

### Step 1

Learn:

- SOLID
- OOP
- UML

### Step 2

Master:

- Strategy
- Factory
- Observer
- Decorator

These appear everywhere in LLD interviews.

### Step 3

Solve 10–15 real systems:

- Parking Lot
- ATM
- Splitwise
- BookMyShow

### Step 4

For every problem:

- Draw UML
- Explain responsibilities
- Identify SOLID principles
- Explain extensibility
- Discuss tradeoffs
